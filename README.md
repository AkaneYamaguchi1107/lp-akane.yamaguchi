# Ussg

Update Static Site Generator.

Gulp / Pug / SCSS / ES6 / Webmanifest

## Usage

```bash
yarn install
gulp
```

## Release

```bash
gulp release
```

## Deploy

### GitHub

```bash
npm run deploy-github
```

### GitLab

Automatically deployed to GitLab Pages.

## Generate Blank Pages

```bash
gulp create-blank-pages
```
